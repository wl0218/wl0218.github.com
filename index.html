Basic Linux Commands

Posted on 2022-09-28 In Study Notes

Shortcuts

Commands

  • pwd

    show the path of the current working directory

  • cd + full path / name of directory

    navigate through files and directories

    • `cd ..`: move one directory up

    • cd: go to the home folder

    • cd -: move to the previous directory

  • ls + full path / name of directory

    list the contents of directory

    • ls -R: list all files in the sub-directories
    • ls -a: show the hidden files
    • ls -al: list the files and directories with detailed information

  • cp

    copy files from the current directory to a different directory

  • mv

    move files

  • mkdir

    make a directory

  • rmdir

    remove a directory (empty dir only)

  • rm

    delete directories and the contents within them

  • touch

    create a new file

  • sudo

    short for SuperUser DO

  • head

    display the first ten lines of a text file

  • tail

    display the last ten lines of a text file

  • clear

    clean out the terminal
